# 🏡 로컬 sLLM 기반 프라이버시 보호형 스마트홈 제어 허브

> **외부 인터넷 연결과 데이터 유출 걱정 없는 100% 온디바이스 스마트홈 제어 시뮬레이터 프로젝트입니다.**  
> 사용자가 자유롭게 입력한 문자 메시지(자연어)를 로컬 sLLM(소형 언어 모델)이 스스로 분석하여, 가전제품이 이해할 수 있는 정형 제어 신호(JSON)로 자동 통역합니다.

<br>

## ✨ 핵심 기능 (Features)
- **100% 오프라인 & 프라이버시 보호**: 외부 가전 제어 API나 클라우드 서버를 일절 사용하지 않아 완벽한 보안 유지.
- **다중 명령 완벽 처리 (Sentence Parsing)**: "에어컨 끄고 로봇청소기로 청소해줘. 조명도 밝혀줘"와 같이 한 번에 여러 개를 명령해도 문장을 자동으로 분리하여 순차 처리.
- **유연한 자연어 예외 처리**: sLLM이 상황에 따라 다르게 출력하는 다양한 어휘(`밝히기`, `청소해줘`, `가동`)를 가전 표준 명령어(`켜기`, `끄기`)로 지능적 맵핑.
- **실시간 대시보드 모니터링**: 웹 환경(Streamlit)에서 변환된 JSON 로그와 우측 가전제품의 상태 변화를 실시간 인터랙티브 UI로 시각화.

<br>

## 🛠️ 기술 스택 (Tech Stack)
- **Frontend / Dashboard**: Streamlit (Python)
- **sLLM Engine**: Ollama (`qwen2.5-1.5b:latest` 또는 `kanana-1.5:latest`)
- **Runtime Environment**: Windows + WSL (Windows Subsystem for Linux : sLLM 서비스용도)
- **Language**: Python 3.11+

<br>

## 🚀 시작하기 (Installation & Setup)

### 1. 로컬 sLLM 서버 구동 (WSL 환경)
WSL 터미널을 열고 백그라운드에 구동 중인 기존 서비스를 확인한 후, 외부 포트 개방 모드로 Ollama를 실행합니다.
```bash
# 기존 시스템 자동 실행 서비스 중지 및 프로세스 해제
sudo systemctl stop ollama
sudo killall ollama

# Windows 호스트와 통신을 위한 포트 개방 및 서버 실행
OLLAMA_HOST=0.0.0.0 ollama serve
```

### 2. 필요한 파이썬 라이브러리 설치 (Windows 환경)
Windows 터미널 또는 가상환경에서 아래 패키지들을 설치합니다.
```bash
pip install streamlit requests
```

### 3. 애플리케이션 실행
소프트웨어가 저장된 디렉토리로 이동한 뒤 Streamlit을 구동합니다.
```bash
streamlit run app.py
```
*실행 시 웹 브라우저(`http://localhost:8501`)가 자동으로 열립니다.*

## 💡 시스템 작동 구조
1. **사용자 입력 수신**: Streamlit UI를 통한 자연어 텍스트 입력.
2. **문장 분할 전처리**: 정규식 및 문자열 가공을 통한 단일 동작 단위 분리.
3. **Ollama JSON 추론**: 쪼개진 개별 문장을 Qwen 2.5 모델에 보내 정형화된 JSON 형태 포맷으로 응답 유도.
4. **표준 어휘 변환**: 가전제품 제어 장치가 인식할 수 있는 고정 명령어로 유연 매핑.
5. **UI 상태 업데이트**: 세션 상태(`st.session_state`) 데이터를 갱신하여 웹 화면에 즉시 피드백 제공.
